yle-dl

Ladda ner videor från Yle Arenan

Koden på Github
Suomeksi
In English

yle-dl är ett verktyg för att ladda ner mediefiler från Yle Arenan, Yle arkivet och Yle nyheter. Dessutom kan du med yle-dl ladda ner Arenans internetradior och direkt TV sändningar.

Du kan se program som laddas ner till din dator även om de redan har lämnat Arenan. När du spelar upp från din egen dator stammar videorna inte heller som när du spelar över en långsam internetanslutning.

De nedladdade programmen sparas i Matroska (.mkv) eller MP4 format. De kan sedan spelas upp med till exempel VLC.

Du kan endast ladda ner videor för eget bruk. Spridning av Arenans program utan tillstånd från upphovsrättsinnehavaren är förbjudet.

Installation

  1. Installera nödvändiga bibliotek och program: Python 3.7+, ffmpeg 4.1+. För att ladda ner vissa Arena program behövs också wget.
  2. Installera pipx
  3. pipx install yle-dl

Användning

yle-dl [alternativ] URL

Några av alternativen:

-o filename
Spara streamen till namngivna filen
--destdir dir
Spara filer till en mapp
--latestepisode
Ladda ner senaste avsnitten från sidan
--showmetadata
Skriv ut streamens metadata, ladda inte ner
--proxy uri
Använd ett SOCKS-proxy för att ladda ner streamens metadata. Exempel: --proxy socks5://localhost:7777

Använd kommandot yle-dl --help för att se alla alternativ.

Till exempel:

yle-dl http://areena.yle.fi/tv/1515302

eller

yle-dl http://areena.yle.fi/tv/1515302 -o /your/video/dir/video.mp4
